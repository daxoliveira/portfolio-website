A platform for NGOs to get helpBe The Hero

A web and mobile app built with React.js and React Native

The goal of this project was to create a platform to help NGO's to share their specific needs in a case by case basis, so individuals could choose which one to help.

The design and conceptualization of the project was made available by an online programming school called Rockseat. Their goal was to give web developers a chance to build a full-stack and mobile application from scratch using Node.js, React.js, and React Native.
