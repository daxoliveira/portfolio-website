Game of Thrones Top Trumps (Super Trunfo)

A game built with HTML, CSS, and JavaScript

A digital version of the card game "Top Trumps (Super Trunfo)" in which a single person plays against the machine. Anyone can play it. You can play anytime as long as you have a computer with internet connection. Also, this version of "Top Trumps" is not timed.

To win the game is very simple, you have to take all of your opponent's cards.
